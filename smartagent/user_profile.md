# Профиль пользователя

## Текущий проект
- **Тип продукта:** Десктопное приложение
- **Назначение:** CRM для стоматологической клиники
- **Текущий этап:** Проектирование экранов
- **Функциональные требования:**
  - Запись
  - Медкарта
  - Расчет суммы
  - Статистика посещений
  - Учет расходников

## Отложенные проекты
### Мобильное приложение
- **Статус:** Отложен
- **Назначение:** Управление диабетом у котов
- **Целевая аудитория:** Владельцы котов-диабетиков
- **Функциональные требования:**
  - Поддержка нескольких котов
  - Отслеживание уровня сахара в крови (с отображением кривой на графике)
  - Предоставление полезных советов
  - График и учет визитов к врачу
  - Дневник записи симптомов
  - Учет текущей дозы инсулина
- **Технические и продуктовые решения:**
  - **Стек технологий:** Kotlin Multiplatform (KMP)
  - **Ввод данных:** Исключительно ручной ввод
  - **Хранение данных:** Локальное
  - **Состав MVP:**
    - **Исключено:** Отрисовка графика уровня сахара в крови
    - **Включено:** Все остальные заявленные функции
