# Профиль пользователя

## Текущие проекты и цели
- **Активный проект 1:** CRM для стоматологической клиники.
  - **Платформа:** Десктопное приложение.
  - **Текущий фокус:** Модуль пациентов (начать работу с него).
- **Активный проект 2:** Мобильное приложение для владельцев котов-диабетиков.
  - **Целевая аудитория:** Владельцы нескольких котов.
  - **Технические требования:** Мультиплатформенное приложение без бэкенда.

## Продуктовые требования и предпочтения
**Мобильное приложение для котов-диабетиков:**
- **Предпочитаемый технологический стек:** KMP + CMP (Kotlin Multiplatform + Compose Multiplatform).
- **Подход к MVP:** Все заявленные функции должны быть включены в минимально жизнеспособный продукт (MVP).
- **Обязательный функционал (5 функций):**
  1. Отслеживание уровня сахара в крови с отображением кривой на графике.
  2. Полезные советы.
  3. График визитов к врачу.
  4. Запись симптомов.
  5. Учет текущей дозы инсулина.
